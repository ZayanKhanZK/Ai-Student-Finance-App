# My AI Finance

```
A C++/Qt Mobile Application powered by Gemini AI
```
## 🚀 Overview

```
My AI Finance is a minimalist, student-focused financial tracker. Instead
of manually filling out tedious forms, you just tell the app what you did in
plain English. Using the Google Gemini API, the app parses your intent,
categorizes the transaction, and updates your balance in real-time.
C++ 17 Qt 6.x QML Gemini AI Android
```
## 🛠️ Features

```
AI Parsing: Convert natural language into structured data.
Persistent Balance: Local storage ensures data survives app restarts.
Modern UI: Minimalist dark mode dashboard built with Qt Quick.
Interactive History: Expandable transaction logs with deletion support.
```
## 🔑 How to get your Gemini API Key

```
The app requires a Google Gemini API key to understand your text inputs.
Follow these steps:
Go to the Google AI Studio.
```
#### 1.


```
Sign in with your Google Account.
Click on "Get API key" in the top left sidebar.
Click "Create API key in new project".
Copy the key. You will need to paste this into the app's settings
menu.
```
```
Security Tip: Never share your API key publicly. Blur the key and your
Project ID in any videos or screenshots.
```
## 📲 Installation Guide (Android)


### Method Build from Source

If you want to modify the code and build it yourself:

```
Install Qt Creator and ensure you have the Android Build Kits
configured.
Clone this repository:
git clone https://github.com/ZayanKhanZK/Ai-
Student-Finance-App.git
```
```
Open CMakeLists.txt in Qt Creator.
```


```
Select the Android arm64-v8a kit.
Connect your phone via USB and press the Green Play Button.
```
## ⚙️ Configuration

```
Launch the app and click Enter Dashboard.
Click the API button in the top right.
Paste your Gemini API Key and hit Save.
Start typing transactions (e.g., "Spent 500 on coffee").
```
